# Objetive 
- The password for the next level is stored in the file data.txt in one of the few human-readable strings, preceded by several ‘=’ characters.


## Solution
- For this level I learned to use the strings command, which filtered the human-readable characters with a minimum of 20 characters with the parameter strings -n 20 data.txt, which filtered the password.
- FGUW5ilLVJrxX9kMYMmlN4MgbpfMiqey

- ssh
- pwd
- ls
- cat 
- sort
- uniq 
- strings -u data.txt
