## Objetive 
-The password for the next level is stored in a hidden file in the inhere directory.

## Solution
-To solve this level, I had to find a hidden file in the `inhere` directory called `...Hiding-From-You`. I used the following command to see the password hidden within it: `cat inhere/...Hiding-From-You (2WmrDFRmJIq3IPxneAaMGhap0pFhF3NJ)` 
## commands used

-ssh 
-pwd 
-ls 
-cd 
-cat 
-cat inhere/...Hiding-From-You
